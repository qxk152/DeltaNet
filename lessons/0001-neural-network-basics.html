神经网络基础

这一课介绍构成 GDN 的基本组件：线性层、激活函数、归一化、残差连接。 这些是所有现代深度学习模型的积木，GDN 的每一层都用到它们。

线性层：最基本的变换

线性层（也叫全连接层）做一件事：矩阵乘法加偏置。

$$y = Wx + b$$

其中 $x \in \mathbb{R}^{d_{in}}$ 是输入，$W \in \mathbb{R}^{d_{out} \times d_{in}}$ 是权重， $b \in \mathbb{R}^{d_{out}}$ 是偏置，$y \in \mathbb{R}^{d_{out}}$ 是输出。

在 GDN 代码中你会看到大量线性层：

self.q_proj = nn.Linear(hidden_size, key_dim, bias=False)   # 投影出 Q
self.k_proj = nn.Linear(hidden_size, key_dim, bias=False)   # 投影出 K
self.v_proj = nn.Linear(hidden_size, value_dim, bias=False)  # 投影出 V
关键直觉：线性层就是"用一个矩阵把一个向量从一种表示空间变换到另一种表示空间"。 Transformer 和 GDN 都先把输入投影成 Q/K/V 三个不同的表示，再在这些表示上做注意力计算。

激活函数：引入非线性

只有线性层的网络，无论堆叠多少层，整体仍是线性的。激活函数引入非线性，使网络能学习复杂模式。

ReLU（最经典）

$$\text{ReLU}(x) = \max(0, x)$$

简单、高效，但负数输入完全被丢弃。

SiLU / Swish（GDN 使用）

$$\text{SiLU}(x) = x \cdot \sigma(x) = \frac{x}{1 + e^{-x}}$$

SiLU 是 $x$ 和 sigmoid 的乘积——负数输入被"压低"而非完全丢弃。 现代模型（Llama、GDN）普遍使用 SiLU。

归一化：稳定训练

归一化层把激活值拉回合理范围，防止数值爆炸或消失。GDN 使用两种归一化：

LayerNorm

对每个样本的所有特征维度做归一化：

$$\text{LayerNorm}(x) = \gamma \cdot \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}} + \beta$$

其中 $\mu, \sigma^2$ 是 $x$ 各维度的均值和方差，$\gamma, \beta$ 是可学习参数。

RMSNorm（GDN 使用）

LayerNorm 的简化版——不减均值，只用均方根归一化：

$$\text{RMSNorm}(x) = \gamma \cdot \frac{x}{\sqrt{\frac{1}{d}\sum_{i} x_i^2 + \epsilon}}$$
为什么 GDN 用 RMSNorm？
  • 计算更快（不需要算均值）
  • 实践中效果与 LayerNorm 相当
  • 在 GDN kernel 中，RMSNorm 常与门控融合在一起（FusedRMSNormGated）

残差连接：让深层网络可训练

残差连接（Residual Connection）简单但关键：

$$y = x + f(x)$$

输出 = 输入 + 子层输出。梯度可以直接通过 $+x$ 流回，避免深层网络的梯度消失。

# GDN 中的残差结构（伪代码）
h = hidden_states
o = attention_layer(h)   # 或 GDN layer
h = h + o                 # 残差连接
o = mlp_layer(h)
h = h + o                 # 又一个残差

Pre-Norm vs Post-Norm

归一化放在残差连接的之前还是之后，影响很大：

# Pre-Norm (GDN/现代模型用)
h = hidden_states + attention(norm1(hidden_states))
h = h + mlp(norm2(h))

# Post-Norm (原始 Transformer)
h = norm1(hidden_states + attention(hidden_states))
h = norm2(h + mlp(h))

Pre-Norm 更容易训练深层模型，是现代标准。

把积木拼起来：一个 Transformer/GDN Block 的结构

有了以上组件，一个完整的 block 长这样：

def block(x):
    # 子层1: 归一化 + 注意力/GDN + 残差
    h = x + gdn_layer(rmsnorm(x))

    # 子层2: 归一化 + MLP + 残差
    out = h + mlp(rmsnorm(h))
    return out
本课要点：GDN 的"新东西"只是把 gdn_layer(...) 内部的注意力换成 gated delta rule。 其余部分（线性投影、SiLU、RMSNorm、残差、MLP）都和标准 Transformer 完全一样。 所以理解 GDN = 理解那个 gdn_layer 内部做了什么。

测验

问题 1：SiLU(x) 的公式是什么？
  • max(0, x)
  • x · σ(x) = x / (1 + e^(-x))
  • x²
  • tanh(x)
问题 2：RMSNorm 与 LayerNorm 的主要区别是？
  • RMSNorm 不使用可学习参数 γ
  • RMSNorm 不减去均值，只用均方根归一化
  • RMSNorm 只对部分维度归一化
  • RMSNorm 使用 sigmoid 而非线性归一化
问题 3：Pre-Norm 残差连接的公式是？
  • h = norm(x + f(x))
  • h = x + f(norm(x))
  • h = norm(x) + f(x)
  • h = norm(f(x + norm(x)))
